A simple plugin to fetch and display lottery results in a WordPress widget.

== Description ==
This plugin allows WordPress users to display the latest lottery results via a shortcode or widget. 
It fetches real-time results from an API and presents them in a customizable format.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[lottery_results]` shortcode to display results.
